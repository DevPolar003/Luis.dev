# Luis.dev - Personal Blog Project 🚀

Este projeto é um blog pessoal dinâmico que utiliza **Spring Boot** no backend para gerenciar postagens e **JavaScript (Vanilla)** no frontend para renderização dinâmica e interatividade.

## 📋 Sobre o Projeto
O Luis.dev foi desenvolvido para praticar a integração entre uma API REST em Java e um frontend moderno. O sistema carrega postagens armazenadas no servidor e as exibe em uma interface responsiva, contando com funcionalidades como:
- Consumo de API dinâmica para listagem de posts.
- Interface responsiva com Bootstrap 5.
- Sistema de Modal para visualização de detalhes.

## 🛠️ Tecnologias Utilizadas

### Backend
- **Java 25**
- **Spring Boot 4.05**
- **Spring Web**

### Frontend
- **HTML5 & CSS3**
- **JavaScript ES6+** 
- **Bootstrap 5.3** 
- **Bootstrap Icons**
